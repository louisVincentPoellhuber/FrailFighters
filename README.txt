Projet fait par Louis-Vincent Poellhuber et Léo Brulotte dans le cadre d'un projet intégrateur au cégep de Bois-de-Boulogne en hivers 2019.
C'est un protoype de jeu de combat en 2D, semblable à Super Smash Brothers.

Le projet utilise la librairie libgdx pour faire le jeu. Je ne suis pas trop sûr de comment nous avons initialisé le projet, mais je suis incapable de faire run 
le FrailFighters.jar situé dans FrailFighters\out\artifacts\FrailFighters_jar. Le projet est capable de run dans un IDE par contre. 

Le prototype n'a jamais été complété, donc une fois une partie lancée, il n'y a pas de menu pour quitter. L'application quitte automatiquement si un joueur 
est éliminé. La plupart du temps de développement a été consacré à transférer nos connaissances de java sur NetBeans à la librairie libgdx, donc il y a peu 
de personnages et de cartes. Le système d'items n'a pas encore été complété, mais ils peuvent être frappés. 

CONTRÔlES
Joueur 1:
Mouvement: WASD
Attaque de côté: Spacebar (maintenir)
Attaque vers le bas: Q (maintenir)
Attaque vers le haut: E (maintenir)

Joueur 1:
Mouvement: Flèches
Attaque de côté: K (maintenir)
Attaque vers le bas: J (maintenir)
Attaque vers le haut: L (maintenir)
